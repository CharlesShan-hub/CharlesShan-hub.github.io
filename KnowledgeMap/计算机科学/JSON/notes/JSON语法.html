主要用法

  • 数组（Array）用方括号表示

  • 对象（Object）用圆括号表示

  • 键值对（Name/Value）组合成数组和对象

  • 名称（Name）置于双引号中，

  • 值（Value）：注意不能有函数和undefined

    • 字符串：注意使用双引号，不能使用单引号

    • 数值：注意

    • null

    • 对象

    • 数组

  • 并列的数据用逗号分隔

  • {
      "string":"Chalres Shan",
      "array":[100,1e-4,
               true,false,
               ["nested"],
               {"test":null}
              ]
    }
